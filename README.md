# 🧬 SMA Biomarker Clustering

Clustering of co-expressed genes relative to known Spinal Muscular Atrophy (SMA) biomarkers (**NfL, HSPA7, SMN2**) using computational gene expression analysis.

---

## 📌 Overview

This project presents a computational pipeline to identify and analyze gene clusters that are co-expressed with known SMA biomarkers. By leveraging correlation-based methods, the workflow groups genes into biologically meaningful clusters and enables the discovery of potential biomarker-associated genes.

---

## 📂 Repository Structure

- `code/` → MATLAB scripts for co-expression analysis and clustering  
- `results/` → Example outputs (cluster visualization and gene lists)

---

## ⚙️ Workflow Overview

1. **Compute co-expression** relative to selected biomarkers (NfL, HSPA7, SMN2)  
2. **Generate gene clusters** based on correlation thresholds  
3. **Analyze clusters** to identify potential biomarker-associated genes

## 🔬 Interpretation

The clustering results reveal distinct groups of genes co-expressed with known SMA biomarkers.  

- **Single-biomarker clusters** (e.g., NfL, HSPA7, SMN2) contain larger gene sets, reflecting broader co-expression patterns.  
- **Multi-biomarker clusters** (e.g., NfL+HSPA7+SMN2) are more selective, indicating stronger and potentially more biologically relevant regulatory relationships.  

These findings suggest that combining biomarkers can help identify more specific gene signatures associated with SMA.

---

## 📥 Input Data

The analysis is based on normalized gene expression data obtained from publicly available datasets. The data is preprocessed to extract expression profiles for SMA-related conditions and corresponding controls.
